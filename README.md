# Trabalho Naruto - Natureza Discreta
## O objetivo deste trabalho é levar para alunos do ensino médio, de escolas públicas e particulares, como a matemática pode ser usada de uma forma descontraída para aprender, aumentar o foco, agilidade e redução em respostas que exigem raciocínio lógico.
### Cada jogador irá escolher um dos dois personagens disponíveis(Sasuke ou Naruto). Ao iniciar a batalha, o jogador terá uma experiência imersiva, que resgata boas memórias do anime, como música de batalha e poderes inesquecíveis. Porém, para que seja possível utilizar cada poder do personagem, o jogador terá que demonstrar sua habilidade "ninja" na matemática, sobre: Tabela Verdade; Conjução; Disjunção; Condicional e Bicondicional.
### O intuito dos criadores é provar que a matemática sempre pode ser divertida, sem limite de idade.
